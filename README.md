# Horizontal vs Vertical Scaling Comparison

## Project Overview

This project compares two cloud scaling techniques:

- Horizontal Scaling
- Vertical Scaling

The project includes:

- Python implementation
- JSON input/output
- Comparison report
- Architecture diagrams
